## 템플릿 2: 행사 안내 (회신 필요)

```
┌─────────────────────────────────────────────────────────┐
│                    가정통신문 제 OOO호                   │
│                                                          │
│                [행사명] 안내 및 참가 동의서               │
└─────────────────────────────────────────────────────────┘

학부모님께

  안녕하십니까?
  가정에 건강과 행복이 가득하시기를 바랍니다.

  우리 학교에서는 [행사명]을 아래와 같이 실시하고자 합니다.
  학부모님의 적극적인 협조 부탁드립니다.

                            - 아 래 -

  1. 행사명: [행사명]
  2. 일  시: YYYY년 MM월 DD일(요일) HH:MM ~ HH:MM
  3. 장  소: [장소]
  4. 대  상: [대상 학년/학생]
  5. 내  용:
     - [프로그램 1]
     - [프로그램 2]
  6. 준비물: [필요시]
  7. 비  용: [비용 발생 시 안내]

  ※ 참가 동의서를 MM월 DD일(요일)까지 담임선생님께 제출해 주시기 바랍니다.

  문의사항은 교무운영부(042-XXX-XXXX)로 연락 주시기 바랍니다.
  감사합니다.

                                        YYYY년 MM월 DD일
                                        대전과학고등학교장

                    [행사명] 참가 동의서

  위 안내 내용을 확인하였으며, 자녀의 행사 참가에 동의합니다.

  □ 참가합니다          □ 참가하지 않습니다

  학생: (   )학년 (   )반 (   )번  성명: ________________

  학부모: ________________ (서명 또는 인)

  연락처: _______________________

## 템플릿 3: 학부모 설명회 안내

```
┌─────────────────────────────────────────────────────────┐
│                    가정통신문 제 OOO호                   │
│                                                          │
│               YYYY학년도 학부모 설명회 안내               │
└─────────────────────────────────────────────────────────┘

학부모님께

  안녕하십니까?
  평소 학교 교육에 보내주시는 관심과 성원에 깊이 감사드립니다.

  우리 학교에서는 학부모님과의 원활한 소통과 학교 교육활동에 대한
  이해를 돕고자 학부모 설명회를 아래와 같이 개최합니다.
  바쁘시더라도 참석하시어 자녀 교육에 도움이 되시기를 바랍니다.

                            - 아 래 -

  1. 일  시: YYYY년 MM월 DD일(요일) HH:MM ~ HH:MM
  2. 장  소: 대전과학고등학교 [장소]
  3. 대  상: [해당 학년] 학부모
  4. 내  용:
     - [주제 1]: HH:MM ~ HH:MM
     - [주제 2]: HH:MM ~ HH:MM
     - 학급별 간담회: HH:MM ~ HH:MM
  5. 기  타:
     - 주차 공간이 협소하오니 가급적 대중교통을 이용해 주시기 바랍니다.
     - 실내화를 지참해 주시기 바랍니다.

  ※ 참석 여부를 MM월 DD일(요일)까지 회신해 주시기 바랍니다.

  문의: 교무운영부 (042-XXX-XXXX)

                                        YYYY년 MM월 DD일
                                        대전과학고등학교장

┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄ ✂ 회 신 란 ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄

                 학부모 설명회 참석 여부 회신

  학생: (   )학년 (   )반 (   )번  성명: ________________

  □ 참석합니다 (참석 인원:    명)

  □ 참석하지 못합니다
    - 사유: _______________________________________________

  학부모: ________________ (서명 또는 인)

                                        YYYY년   월   일

## 작성 시 유의사항

### 기본 원칙
- 간결하고 명확하게 작성
- 육하원칙(누가, 언제, 어디서, 무엇을, 왜, 어떻게) 준수
- 존댓말 사용, 학부모 존중

### 필수 확인 사항
- [ ] 가정통신문 번호 확인
- [ ] 날짜 정확성
- [ ] 연락처 정확성
- [ ] 회신 기한 명시 (필요시)
- [ ] 절취선/회신란 포함 (필요시)

### 발송 방법
- 종이: 학생 편 배부
- 전자: e알리미, 학교 홈페이지
