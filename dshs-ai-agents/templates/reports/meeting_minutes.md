## 템플릿 2: 학교운영위원회 회의록

```
┌─────────────────────────────────────────────────────────┐
│       YYYY학년도 제O차 학교운영위원회 회의록             │
└─────────────────────────────────────────────────────────┘

1. 회의 개요
   - 일  시: YYYY년 MM월 DD일(요일) HH:MM ~ HH:MM
   - 장  소: 대전과학고등학교 [회의실명]
   - 회의 종류: □ 정기회  □ 임시회

2. 참석 현황
   | 구분 | 성명 | 참석 | 비고 |
   |------|------|------|------|
   | 위원장 | OOO | O | |
   | 학부모위원 | OOO | O | |
   | 학부모위원 | OOO | O | |
   | 교원위원 | OOO | O | |
   | 교원위원 | OOO | X | 출장 |
   | 지역위원 | OOO | O | |
   | 당연직 | OOO (교장) | O | |

   ※ 참석 N명 / 불참 N명 (정족수 충족 여부: O/X)

3. 안건

┌──────┬──────────────────────────────┬────────┐
│ 번호 │ 안건명                        │ 구분   │
├──────┼──────────────────────────────┼────────┤
│ 1    │ [안건명]                      │ 심의   │
│ 2    │ [안건명]                      │ 자문   │
│ 3    │ [안건명]                      │ 보고   │
└──────┴──────────────────────────────┴────────┘

4. 심의(자문) 내용

【제1호 안건】 [안건명] - 심의

  ▷ 제안 설명 (OOO)
    - [제안 내용 상세 설명]

  ▷ 질의 및 응답
    Q. (OOO 위원) [질의 내용]
    A. (OOO) [응답 내용]

  ▷ 토의 내용
    - (OOO 위원) [의견]
    - (OOO 위원) [의견]

  ▷ 의결
    - 결과: 원안 가결 (찬성 N, 반대 N, 기권 N)
    - 조건/부대의견: [있을 경우 기재]

【제2호 안건】 [안건명] - 자문
  [상동 형식]

【제3호 안건】 [안건명] - 보고
  - [보고 내용 요약]
  - [위원 질의 및 의견]

5. 기타 안건 / 건의사항
   - [내용]

6. 차기 회의 안내
   - 예정일: YYYY년 MM월 중
   - 예정 안건: [안건]

7. 폐회 선언: HH:MM

                    작성일: YYYY년 MM월 DD일
                    작성자: OOO (간사)
                    확  인: OOO (위원장)
```

## 작성 시 유의사항

### 기본 원칙
- 객관적 사실에 기반하여 작성
- 발언자 명시 (필요시)
- 의결 결과 명확히 기록
- 결정 사항의 담당자와 기한 명시

### 보안 등급
- 인사자문위원회: 비공개
- 학교운영위원회: 원칙적 공개 (개인정보 제외)
- 일반 회의: 내부 공유

### 필수 확인 사항
- [ ] 참석자 명단 정확성
- [ ] 안건별 의결 결과 기록
- [ ] 결정 사항 담당자/기한 명시
- [ ] 작성자/확인자 서명
