## 템플릿 1: 입학식 계획서

```
┌─────────────────────────────────────────────────────────┐
│           YYYY학년도 입학식 계획(안)                     │
└─────────────────────────────────────────────────────────┘

1. 목적
   - 신입생의 입학을 축하하고 학교 구성원으로서의 소속감 고취
   - 학교의 교육 목표와 비전 공유
   - 신입생과 학부모에게 학교 교육활동 안내

2. 개요
   - 일 시: YYYY년 MM월 DD일(요일) HH:MM
   - 장 소: 대전과학고등학교 [장소]
   - 대 상: 제OO회 신입생 및 학부모
   - 참석 예상 인원: 학생 OOO명, 학부모 OOO명

3. 세부 일정

| 시간 | 내용 | 담당 | 비고 |
|------|------|------|------|
| HH:MM~HH:MM | 등록 및 착석 | 1학년부 | |
| HH:MM~HH:MM | 개식 선언 | 사회자 | |
| HH:MM~HH:MM | 국민의례 | 전체 | |
| HH:MM~HH:MM | 신입생 입장 | 1학년부 | 반별 입장 |
| HH:MM~HH:MM | 입학 허가 선언 | 교장 | |
| HH:MM~HH:MM | 교장 훈화 | 교장 | |
| HH:MM~HH:MM | 신입생 선서 | 대표학생 | |
| HH:MM~HH:MM | 재학생 환영사 | 학생회장 | |
| HH:MM~HH:MM | 학부모 대표 인사 | 학부모 대표 | |
| HH:MM~HH:MM | 교직원 소개 | 교감 | |
| HH:MM~HH:MM | 폐식 | 사회자 | |
| HH:MM~HH:MM | 학급별 이동 | 담임 | 담임 첫 만남 |

4. 업무 분장

| 업무 | 담당 부서 | 담당자 | 세부 내용 |
|------|----------|--------|----------|
| 총괄 | 교무운영부 | | 전체 진행 총괄 |
| 식순 진행 | 교무운영부 | | 사회, 시간 관리 |
| 신입생 안내 | 1학년부 | | 등록, 착석 안내 |
| 학부모 안내 | 교무운영부 | | 좌석 안내, 주차 |
| 음향/영상 | 교육정보부 | | 마이크, 스크린 |
| 무대 설치 | 방과후인성부 | | 단상, 현수막 |
| 기념 촬영 | | | 단체 사진 |

5. 준비물

| 항목 | 수량 | 담당 | 확인 |
|------|------|------|------|
| 현수막 | 1 | | □ |
| 식순표 | OOO | | □ |
| 교가 악보/음원 | 1 | | □ |
| 신입생 명찰 | OOO | | □ |
| 좌석 배치도 | 1 | | □ |
| 국기 게양 | 1 | | □ |

6. 예산(안)

| 항목 | 금액(원) | 비고 |
|------|---------|------|
| 현수막 제작 | | |
| 행사 용품 | | |
| 기념품 | | |
| 기타 | | |
| **합계** | | |

7. 유의사항
   - 코로나19 등 감염병 상황 시 대응 방안 수립
   - 우천 시 대체 장소 확보
   - 학부모 주차 안내
   - 사진/영상 촬영 동의 안내

8. 기대 효과
   - 신입생의 학교 적응 지원
   - 학교 구성원으로서의 자부심 고취
   - 학부모와의 첫 만남을 통한 신뢰 구축

                                    작성: 교무운영부
                                    작성일: YYYY.MM.DD
```

## 템플릿 2: 졸업식 계획서

```
┌─────────────────────────────────────────────────────────┐
│           YYYY학년도 졸업식 계획(안)                     │
└─────────────────────────────────────────────────────────┘

1. 목적
   - 졸업생의 학업 완성을 축하하고 새로운 출발을 격려
   - 학교생활의 마무리와 추억 공유
   - 졸업생, 재학생, 학부모 간 유대 강화

2. 개요
   - 일 시: YYYY년 MM월 DD일(요일) HH:MM
   - 장 소: 대전과학고등학교 [장소]
   - 졸업생: 제OO회 졸업생 OOO명

3. 세부 일정

| 시간 | 내용 | 담당 | 비고 |
|------|------|------|------|
| HH:MM~HH:MM | 등록 및 착석 | 3학년부 | |
| HH:MM~HH:MM | 개식 선언 | 사회자 | |
| HH:MM~HH:MM | 국민의례 | 전체 | |
| HH:MM~HH:MM | 졸업장 수여 | 교장 | 대표 수여 |
| HH:MM~HH:MM | 각종 상장 수여 | 교장 | |
| HH:MM~HH:MM | 공로상 수여 | 교장 | 교직원 포함 |
| HH:MM~HH:MM | 교장 훈화 | 교장 | |
| HH:MM~HH:MM | 내빈 축사 | 내빈 | |
| HH:MM~HH:MM | 졸업생 대표 답사 | 대표학생 | |
| HH:MM~HH:MM | 재학생 송사 | 대표학생 | |
| HH:MM~HH:MM | 졸업생 노래 | 졸업생 | |
| HH:MM~HH:MM | 폐식 | 사회자 | |

4. 시상 내역

| 상 명 | 수상자 수 | 시상자 | 비고 |
|------|----------|--------|------|
| 대통령상 | 1 | | |
| 교육부장관상 | 1 | | |
| 교육감상 | 1 | | |
| 교장상 | N | | |
| 공로상 | N | | 학생, 교직원 |

5. 업무 분장
   [입학식과 유사한 형식]

6. 특이사항
   - 졸업앨범 배부
   - 졸업생 후배에게 한마디 영상
   - 감사 편지 전달 행사
   - 담임교사 마지막 인사

                                    작성: 교무운영부
                                    작성일: YYYY.MM.DD
```

## 템플릿 3: 학부모의 날 계획서

```
┌─────────────────────────────────────────────────────────┐
│           YYYY학년도 학부모의 날 계획(안)                │
└─────────────────────────────────────────────────────────┘

1. 목적
   - 학부모의 학교 교육활동 참여 기회 제공
   - 학교와 가정의 소통 강화
   - 교육 공동체로서의 협력 관계 구축

2. 개요
   - 일 시: YYYY년 MM월 DD일(요일) HH:MM ~ HH:MM
   - 장 소: 대전과학고등학교 각 교실 및 특별실
   - 대 상: 전 학년 학부모

3. 프로그램

| 시간 | 프로그램 | 장소 | 내용 |
|------|---------|------|------|
| HH:MM~HH:MM | 등록 및 접수 | 현관 | |
| HH:MM~HH:MM | 수업 참관 | 각 교실 | 1~2교시 수업 |
| HH:MM~HH:MM | 학교 설명회 | 강당 | 학교 현황 안내 |
| HH:MM~HH:MM | 점심 식사 | 식당 | 급식 체험 |
| HH:MM~HH:MM | 학년별 간담회 | 각 장소 | 학년 현안 논의 |
| HH:MM~HH:MM | 학급별 상담 | 각 교실 | 담임 상담 |

4. 세부 계획

가. 수업 참관
   - 1교시: [교과] / 2교시: [교과]
   - 참관 시 유의사항 안내

나. 학교 설명회
   - 학교장 인사
   - 교육과정 안내 (교육과정부)
   - 진로진학 안내 (진로진학부)
   - 생활지도 안내 (학생생활안전부)

다. 학년별 간담회
   - 학년별 주요 현안 공유
   - 학부모 의견 수렴
   - Q&A

5. 업무 분장

| 업무 | 담당 부서 | 세부 내용 |
|------|----------|----------|
| 총괄 | 교무운영부 | 전체 기획 및 진행 |
| 등록/안내 | 각 학년부 | 학부모 등록 및 안내 |
| 수업 | 해당 교과 | 공개 수업 운영 |
| 급식 | 교무운영부 | 식당 협조, 배식 |
| 주차 | 교무운영부 | 주차 안내 |

6. 예산

7. 안내 및 홍보
   - 가정통신문 발송: MM월 DD일
   - 참석 희망 조사: MM월 DD일까지
   - 문자 안내: 행사 전일

                                    작성: 교무운영부
                                    작성일: YYYY.MM.DD
```
