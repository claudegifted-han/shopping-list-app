## 자주 쓰는 명령어

### 문서 작성
```
@교무운영부 [제목] 공문 작성해줘
@교무운영부 [제목] 가정통신문 작성해줘
@교무운영부 [회의명] 회의록 정리해줘
```

### 자율연구
```
@영재교육부 [학년] 팀 현황 알려줘
@영재교육부 [학생] NEIS 기재문 작성해줘
@영재교육부 구술시험 일정표 만들어줘
```

### 성적 분석
```
@교육평가부 [과목] 성적 분석해줘
@교육평가부 시험 일정표 만들어줘
```

### 다부서 협의
```
@브레인스토밍 [정책안] 검토해줘
@전략기획실 [주제] 종합 보고서 작성해줘
```

### 번역
```
@국제교류부 [내용] 영어로 번역해줘
@국제교류부 [영문] 교정해줘
```
