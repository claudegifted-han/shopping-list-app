## 1교시: 시스템 이해 (60분)

### 1.1 AI 에이전트 시스템이란? (10분)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  "1인 교무실" - 16개 부서 AI 부장이 업무를 지원합니다           │
│                                                                 │
│  • 공문, 보고서 자동 작성                                       │
│  • 통계 분석 및 시각화                                          │
│  • 다부서 협의 시뮬레이션                                       │
│  • 정보 검색 및 정리                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**AI가 하는 일:**
- ✅ 문서 초안 작성
- ✅ 데이터 분석 및 정리
- ✅ 일정 관리 지원
- ✅ 정보 검색 및 요약

**AI가 하지 않는 일:**
- ❌ 최종 의사결정
- ❌ 성적 입력/수정
- ❌ 개인정보 직접 열람
- ❌ 대외 문서 발송

### 1.3 사용 방법 (15분)

#### 기본 문법

```
@[부서명] [요청 내용]
```

#### 좋은 요청 예시

```
✅ @교무운영부 3월 15일 교원연수 안내 공문 작성해줘.
   일시는 14:00, 장소는 대강당, 주제는 AI 활용 교육이야.

✅ @영재교육부 1학년 기초연구 팀 현황 표로 정리해줘.

✅ @교육평가부 물리학Ⅰ 중간고사 점수 분석해줘.
   점수: 85, 92, 78, 65, 88, 91, 73, 82, 95, 70
```

#### 피해야 할 요청

```
❌ 공문 작성해줘 (부서 미지정)

❌ @영재교육부 연구 (구체적 내용 없음)

❌ @교육평가부 김OO 학생 성적 알려줘 (개인정보 요청)
```

## 2교시: 실습 (60분)

### 실습 1: 공문 작성 (15분)

**과제:** 교원연수 안내 공문 작성

```
@교무운영부 다음 내용으로 공문 작성해줘:
- 수신: 전 교원
- 제목: 2026학년도 1학기 전문성 신장 연수 안내
- 일시: 3월 20일 15:00
- 장소: 과학관 세미나실
- 주제: 탐구 수업 설계 방법론
```

**확인 사항:**
- [ ] 문서번호 형식 확인
- [ ] 필수 항목 포함 여부
- [ ] 문체 적절성

### 실습 2: 데이터 분석 (15분)

**과제:** 성적 데이터 분석

```
@교육평가부 다음 점수를 분석해줘:
과목: 화학Ⅰ
시험: 1학기 중간고사
점수: 78, 82, 91, 65, 88, 74, 95, 80, 86, 72,
      89, 77, 83, 90, 68, 85, 79, 93, 71, 87
```

**확인 사항:**
- [ ] 평균, 표준편차 확인
- [ ] 분포 시각화 확인
- [ ] 분석 소견 활용 가능성

### 실습 3: 브레인스토밍 (15분)

**과제:** 정책 시뮬레이션

```
@브레인스토밍 다음 정책안을 검토해줘:
현재: 휴대폰 수업시간 사용 금지
변경안: 교사 허가 시 학습 목적 사용 허용
```

**확인 사항:**
- [ ] 다부서 관점 포함 여부
- [ ] 찬반 의견 균형
- [ ] 실행 권고안 유용성

### 실습 4: 자유 실습 (15분)

본인 업무에 필요한 요청을 자유롭게 시도해 보세요.

**실습 아이디어:**

## 부록: 빠른 참조 카드

### 자주 쓰는 명령어

```
# 문서 작성
@교무운영부 [제목] 공문 작성해줘
@교무운영부 [제목] 가정통신문 작성해줘
@교무운영부 [회의명] 회의록 정리해줘

# 자율연구
@영재교육부 [학년] 팀 현황 알려줘
@영재교육부 [학생] NEIS 기재문 작성해줘

# 성적 분석
@교육평가부 [과목] 성적 분석해줘

# 다부서 협의
@브레인스토밍 [정책안] 검토해줘
@전략기획실 [주제] 종합 보고서 작성해줘

# 번역
@국제교류부 [내용] 영어로 번역해줘
```

### 문제 발생 시

```
1. 에이전트가 응답하지 않음
   → 부서명 정확히 입력했는지 확인

2. 원하는 결과가 아님
   → 더 구체적으로 요청 (일시, 대상, 형식 등 명시)

3. 보안 오류
   → 해당 정보 접근 권한 없음 (정당한 절차 이용)

4. 기술적 문제
   → 교육정보부 문의 (042-863-7810)
```
