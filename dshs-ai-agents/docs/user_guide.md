## 1. 시스템 소개

### 1.1 DSHS AI 교무 행정 시스템이란?

대전과학고등학교의 16개 부서 업무를 AI가 지원하는 "1인 교무실" 시스템입니다.

**주요 기능:**
- 📄 공문, 가정통신문, 보고서 자동 작성
- 📊 통계 분석 및 시각화
- 📅 일정 관리 및 충돌 검사
- 🧠 다부서 정책 시뮬레이션
- 📝 NEIS 기재문 생성

### 1.2 에이전트 구성

| 그룹 | 에이전트 | 주요 업무 |
|------|----------|----------|
| CEO | 전략기획실 | 전체 조율, 종합 보고 |
| 교학운영 | 교무운영부, 교육과정부, 교육평가부, 교육연구부, 교육정보부 | 학사 행정 |
| 학생지원 | 학생생활안전부, 방과후인성부, 진로진학부, 사감부, 1·2·3학년부 | 학생 지원 |
| 연구입학 | 과학교육부, 영재교육부, 국제교류부, 입학지원부 | 연구·입학 |
| 특수 | 브레인스토밍 | 다부서 협의 |

## 2. 시작하기

### 2.1 접속 방법

1. Claude Code CLI 실행
2. 프로젝트 디렉토리로 이동
   ```bash
   cd dshs-ai-agents
   ```
3. 에이전트 호출 시작

### 2.2 첫 번째 명령어

```
@교무운영부 안녕하세요, 어떤 업무를 도와드릴 수 있는지 알려주세요.
```

## 3. 에이전트 호출 방법

### 3.1 단일 에이전트 호출

```
@[부서명] [요청 내용]
```

**예시:**
```
@영재교육부 기초 자율연구 팀 조직 현황 정리해줘
@교무운영부 인사자문위원회 회의록 작성해줘
@교육평가부 1학기 중간고사 성적 분포 분석해줘
```

### 3.2 브레인스토밍 (다부서 협의)

```
@브레인스토밍 [정책/안건]
```

**예시:**
```
@브레인스토밍 야간자율학습 시간 단축안 검토해줘
@브레인스토밍 기숙사 외출 규정 변경에 대해 분석해줘
```

### 3.3 전략기획실 (전체 조율)

```
@전략기획실 [종합 업무]
```

**예시:**
```
@전략기획실 올해 학교교육계획 초안 작성해줘
@전략기획실 신학년 준비 현황 전 부서 점검해줘
```

### 3.4 호출 팁

| 좋은 예 | 나쁜 예 |
|---------|---------|
| @교무운영부 3월 교직원 회의 안건 정리해줘 | 회의 안건 정리해줘 (부서 미지정) |
| @영재교육부 2학년 심화연구 팀 현황 알려줘 | @영재교육부 연구 (구체적 요청 없음) |
| @교육평가부 물리학Ⅰ 중간고사 성적 분석해줘 | 성적 분석 (과목/시험 미지정) |

## 4. 부서별 에이전트 안내

### 4.1 교무운영부 (@교무운영부)

**담당 업무:**
- 공문 작성
- 회의록 정리
- 학사일정 관리
- 규정 안내

**자주 쓰는 명령어:**
```
@교무운영부 [주제] 관련 공문 작성해줘
@교무운영부 [회의명] 회의록 정리해줘
@교무운영부 [규정] 관련 조항 알려줘
```

### 4.2 영재교육부 (@영재교육부)

**담당 업무:**
- 자율연구 관리
- 팀 조직
- 발표 일정
- NEIS 기재

**자주 쓰는 명령어:**
```
@영재교육부 [학년] 자율연구 팀 현황 알려줘
@영재교육부 [학생명] NEIS 기재문 작성해줘
@영재교육부 구술시험 일정표 만들어줘
```

### 4.3 교육평가부 (@교육평가부)

**담당 업무:**
- 성적 분석
- 시험 일정
- 평가 계획

**자주 쓰는 명령어:**
```
@교육평가부 [과목] [시험] 성적 분석해줘
@교육평가부 시험 일정표 만들어줘
@교육평가부 성적 분포 시각화해줘
```

### 4.4 브레인스토밍 (@브레인스토밍)

**담당 업무:**
- 정책 시뮬레이션
- 다부서 의견 수렴
- 아이디어 발산

**시뮬레이션 모드:**
| 모드 | 참여 부서 | 소요 시간 |
|------|----------|----------|
| Quick | 3개 | ~2분 |
| Standard | 7개 | ~5분 |
| Deep | 16개 | ~10분 |

**명령어:**
```
@브레인스토밍 [안건] (기본: Standard)
@브레인스토밍 [안건] Quick 모드로
@브레인스토밍 [안건] Deep 분석해줘
```

### 4.5 진로진학부 (@진로진학부)

**담당 업무:**
- 대학 정보 제공
- 진학 상담 지원
- 멘토링 관리

**자주 쓰는 명령어:**
```
@진로진학부 [대학] [학과] 입시 정보 알려줘
@진로진학부 특기자 전형 대비 방법 안내해줘
```

### 4.6 국제교류부 (@국제교류부)

**담당 업무:**
- 번역 (한↔영)
- 국제교류 문서
- 영문 교정

**자주 쓰는 명령어:**
```
@국제교류부 [내용] 영어로 번역해줘
@국제교류부 [영문] 한국어로 번역해줘
@국제교류부 [영문] 교정해줘
```

## 5. 자주 쓰는 업무

### 5.1 공문 작성

```
@교무운영부 다음 내용으로 공문 작성해줘:
- 수신: 전 교원
- 제목: 2026학년도 1학기 교원연수 안내
- 일시: 3월 15일 14:00
- 장소: 과학관 대강당
- 주제: AI 활용 교육 방법론
```

### 5.2 가정통신문 작성

```
@교무운영부 현장체험학습 안내 가정통신문 작성해줘:
- 일시: 4월 10일
- 장소: 한국과학기술원
- 대상: 1학년 전체
- 준비물: 필기구, 점심 도시락
```

### 5.3 회의록 정리

```
@교무운영부 다음 회의 내용 정리해줘:
- 회의명: 3월 교무회의
- 일시: 2026-03-10 16:00
- 참석: 교장, 교감, 부장교사 16명
- 안건: 1) 신학기 학사일정, 2) 자율연구 운영계획
```

### 5.4 성적 분석

```
@교육평가부 다음 점수를 분석해줘:
과목: 화학Ⅰ
점수: 85, 92, 78, 65, 88, 91, 73, 82, 95, 70
```

### 5.5 NEIS 기재문 작성

```
@영재교육부 다음 내용으로 NEIS 자율연구 기재문 작성해줘:
- 학생: 홍길동 (2학년)
- 연구제목: 그래핀 기반 나노소재의 열전도 특성 연구
- 기간: 2026.03 ~ 2026.12
- 내용: 그래핀의 열전도 메커니즘 분석 및 나노복합소재 개발
```

### 5.6 정책 검토

```
@브레인스토밍 다음 정책 변경안 검토해줘:
- 현행: 야간자율학습 19:30~22:30 (3시간)
- 변경안: 20:00~22:00 (2시간)
- 목적: 학생 자율성 증대, 수면시간 확보
```

## 6. 워크플로우 활용

### 6.1 자율연구 사이클

1. **팀 구성** (3월 2주)
   ```
   @영재교육부 1학년 기초연구 팀 구성표 만들어줘
   ```

2. **계획서 검토** (3월 4주)
   ```
   @영재교육부 연구계획서 양식 제공해줘
   ```

3. **중간발표** (6월)
   ```
   @영재교육부 중간발표 일정표 만들어줘
   ```

4. **NEIS 기재** (학기말)
   ```
   @영재교육부 [학생명] NEIS 기재문 작성해줘
   ```

### 6.2 정기고사 운영

1. **시험 범위 공지**
   ```
   @교육과정부 1학기 중간고사 시험 범위 안내문 만들어줘
   ```

2. **시험 일정표**
   ```
   @교육평가부 중간고사 시간표 만들어줘
   ```

3. **성적 분석**
   ```
   @교육평가부 중간고사 전체 성적 분석 보고서 작성해줘
   ```

### 6.3 다부서 협의 → 정책 결정

1. **의견 수렴**
   ```
   @브레인스토밍 [정책안] 검토해줘
   ```

2. **종합 보고서**
   ```
   @전략기획실 [정책안] 종합 분석 보고서 작성해줘
   ```

3. **공문 발송**
   ```
   @교무운영부 [결정사항] 안내 공문 작성해줘
   ```

## 7. 주의사항

### 7.1 보안 규칙

| 데이터 유형 | 보안 등급 | 주의사항 |
|------------|----------|----------|
| 학생 성적 | 🔴 최고기밀 | 개인 성적 직접 요청 불가 |
| 입학전형 | 🔴 최고기밀 | 전형 문제/평가 기준 비공개 |
| 상담 기록 | 🔴 최고기밀 | 개인 상담 내용 접근 불가 |
| 교원 인사 | 🟠 기밀 | 인사 정보 제한적 접근 |
| 일반 학사 | 🟢 일반 | 자유롭게 활용 가능 |

### 7.2 의사결정 권한

AI 에이전트는 **자문 역할**입니다. 다음 사항은 반드시 교사/관리자가 결정:

- ❌ 학생 징계/포상 결정
- ❌ 성적 입력/수정
- ❌ 예산 집행 승인
- ❌ 대외 공문 발송
- ❌ 개인정보 열람

### 7.3 출력물 검토

AI 생성 문서는 반드시 **검토 후 사용**하세요:

1. 사실 관계 확인
2. 날짜/수치 검증
3. 형식 적합성 확인
4. 보안 사항 점검

## 8. 문제 해결

### 8.1 에이전트가 응답하지 않을 때

1. 부서명 정확히 입력했는지 확인
   - ⭕ @영재교육부
   - ❌ @영재교육 (부 누락)

2. 요청 내용이 명확한지 확인
   - ⭕ "1학년 기초연구 팀 현황 알려줘"
   - ❌ "연구" (불명확)

### 8.2 잘못된 정보가 출력될 때

1. 더 구체적인 정보 제공
2. 참조할 데이터/규정 명시
3. 예시 형식 제시

### 8.3 형식이 맞지 않을 때

```
@[부서] 다음 형식으로 작성해줘:
[원하는 형식 예시]
```

### 8.4 보안 오류가 발생할 때

해당 정보 접근 권한이 없는 경우입니다.
- 교육평가부에 성적 데이터 요청
- 입학지원부에 전형 정보 요청

## 9. FAQ

### Q1. 어떤 에이전트를 사용해야 할지 모르겠어요.

**A:** `@전략기획실`에 질문하세요. 적합한 부서를 안내해 드립니다.

```
@전략기획실 자율연구 관련 업무는 어느 부서에 문의해야 하나요?
```

### Q2. 여러 부서 협조가 필요한 업무는요?

**A:** `@브레인스토밍`을 사용하세요. 관련 부서 의견을 종합합니다.

```
@브레인스토밍 신입생 오리엔테이션 프로그램 기획안 검토해줘
```

### Q3. 영어 문서 작성이 필요해요.

**A:** `@국제교류부`가 번역과 영문 작성을 지원합니다.

```
@국제교류부 학교 소개문을 영어로 작성해줘
```

### Q4. NEIS 기재 글자 수가 초과됐어요.

**A:** `@영재교육부`가 글자 수에 맞게 조정해 드립니다.

```
@영재교육부 다음 기재문을 500자 이내로 요약해줘:
[기재문]
```

### Q5. 통계 분석 그래프가 필요해요.

**A:** `@교육평가부`가 분석과 시각화를 지원합니다.

```
@교육평가부 [데이터]를 그래프로 시각화해줘
```
