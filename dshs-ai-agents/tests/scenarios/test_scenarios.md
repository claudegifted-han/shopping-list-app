# DSHS AI 에이전트 테스트 시나리오

## 테스트 개요

### 목적
각 에이전트의 실제 업무 수행 능력을 검증하고 프롬프트를 개선한다.

### 테스트 환경
- Claude Code CLI
- Notion MCP 연동 (선택)
- 로컬 템플릿/데이터 파일

### 평가 기준
| 항목 | 배점 | 설명 |
|------|------|------|
| 정확성 | 30% | 업무 내용의 정확도 |
| 형식 준수 | 25% | 템플릿/양식 준수 여부 |
| 응답 시간 | 15% | 합리적인 응답 시간 |
| 사용성 | 15% | 출력물의 실용성 |
| 보안 준수 | 15% | 민감 정보 보호 |

## 시나리오 1: 영재교육부 - 자율연구 팀 조직

### 테스트 명령
```
@영재교육부 1학년 기초 자율연구 팀 구성표 만들어줘.
학생 수는 90명이고, 분야별로 고르게 배분해줘.
```

### 기대 결과
1. 분야별 균형 잡힌 팀 배분
2. 2~4인 팀 구성
3. 팀 번호, 분야, 인원 표 형식

### 체크리스트
- [ ] 총 인원 90명 맞는지 확인
- [ ] 팀당 2~4인 구성
- [ ] 6개 분야 균형 배분
- [ ] 표 형식 출력

### 예상 출력 형식
```
2026학년도 1학년 기초 자율연구 팀 구성표
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
총 학생: 90명 | 총 팀: 30팀

분야별 배분:
| 분야 | 팀 수 | 인원 |
|------|-------|------|
| 수학 | 5 | 15 |
| 물리 | 5 | 15 |
| 화학 | 5 | 15 |
| 생물 | 5 | 15 |
| 지구과학 | 5 | 15 |
| 정보과학 | 5 | 15 |
```

### 테스트 명령
```
@교무운영부 교원 연수 안내 공문 작성해줘.
```

### 체크리스트
- [ ] 문서번호 형식 (대전과학고-XXXX)
- [ ] 수신: 교원
- [ ] 제목 명확
- [ ] 본문에 일시/장소/주제 포함
- [ ] 교육적 문체
