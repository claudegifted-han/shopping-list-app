# 대전과학고등학교 자율연구 운영 지침

## 제1장 총칙

### 제1조 (목적)
이 지침은 대전과학고등학교 학생들의 자율연구 활동에 관한 사항을 규정함을 목적으로 한다.

### 제2조 (적용대상)
본교 재학생 전원

### 제3조 (연구 유형)
1. **기초 자율연구** (1학년)
   - 과학탐구 기초 역량 배양
   - 2~4인 팀 연구
   - 1학기 수행

2. **심화 자율연구** (2학년)
   - 심층적 연구 수행
   - 개인 또는 2~3인 팀
   - 연간 수행

3. **졸업논문연구** (3학년)
   - 독립적 연구 수행
   - 원칙적 개인 연구
   - 논문 작성 및 구술시험

## 제2장 연구 수행

### 제4조 (팀 구성)
1. 기초연구: 2~4인 (이과 분야별 구성)
2. 심화연구: 1~3인 (주제별 자율 구성)
3. 졸업논문: 원칙적 1인 (공동연구 별도 심사)

### 제5조 (지도교사)
1. 본교 교사 1인 필수
2. 외부 멘토 1인 추가 가능
3. 지도교사 변경: 영재교육부 승인 필요

### 제6조 (연구 분야)
| 분야 | 세부 영역 |
|------|----------|
| 수학 | 순수수학, 응용수학, 통계 |
| 물리 | 역학, 전자기, 양자, 광학 |
| 화학 | 유기, 무기, 물리화학, 생화학 |
| 생물 | 분자생물, 생태, 생명공학 |
| 지구과학 | 대기, 해양, 천문, 지질 |
| 정보과학 | 알고리즘, AI, 데이터과학 |
| 융합 | 2개 이상 분야 융합 |

### 제7조 (연구 일정)

#### 기초연구 (1학년)
| 시기 | 활동 |
|------|------|
| 3월 2주 | 팀 구성 |
| 3월 4주 | 연구계획서 제출 |
| 4월~5월 | 연구 수행 |
| 6월 2주 | 중간발표 |
| 6월~7월 초 | 연구 수행 |
| 7월 1주 | 최종발표 |

#### 심화연구 (2학년)
| 시기 | 활동 |
|------|------|
| 3월 2주 | 팀 구성 |
| 3월 4주 | 연구계획서 제출 |
| 4월~6월 | 1학기 연구 |
| 6월 2주 | 중간발표 |
| 9월~11월 | 2학기 연구 |
| 12월 2주 | 최종발표 |

#### 졸업논문 (3학년)
| 시기 | 활동 |
|------|------|
| 3월 4주 | 주제 확정 |
| 4월 3주 | 계획발표 |
| 5월~8월 | 연구 수행 |
| 9월 2주 | 중간발표 |
| 10월~11월 | 논문 작성 |
| 11월 4주 | 논문 제출 |
| 12월 2주 | 구술시험 |

## 제3장 연구계획서

### 제8조 (계획서 구성)
1. 연구 제목
2. 연구 배경 및 필요성
3. 연구 목표
4. 연구 방법
5. 연구 일정
6. 기대 효과
7. 참고문헌

### 제9조 (계획서 심사)
1. 지도교사 사전 검토
2. 영재교육부 심사
3. 수정보완 요청 시 1주 내 재제출

## 제4장 연구 발표

### 제10조 (발표 형식)
1. **계획발표**: 5분 발표 + 5분 질의
2. **중간발표**: 10분 발표 + 5분 질의
3. **최종발표**: 15분 발표 + 10분 질의
4. **구술시험**: 20분 발표 + 20분 질의

### 제11조 (발표 자료)
1. PPT 또는 PDF 형식
2. 표지에 제목, 학생명, 지도교사 기재
3. 발표 1주 전 제출

### 제12조 (평가 기준)
| 항목 | 배점 |
|------|------|
| 연구 주제의 창의성 | 20% |
| 연구 방법의 적절성 | 20% |
| 연구 수행의 성실성 | 20% |
| 결과의 타당성 | 20% |
| 발표력 및 질의응답 | 20% |

## 제5장 졸업논문

### 제13조 (논문 형식)
1. A4 규격, 한글 또는 영문
2. 분량: 20페이지 이상
3. 구성: 초록, 서론, 본론, 결론, 참고문헌

### 제14조 (구술시험)
1. 심사위원: 지도교사 포함 3인
2. 시간: 40분 (발표 20분 + 질의 20분)
3. 평가: Pass/Fail
4. 재시험: 1회 허용

### 제15조 (논문 제출)
1. 인쇄본 3부 + 전자파일
2. 표절검사 결과 첨부
3. 지도교사 서명 필수

## 제6장 연구윤리

### 제16조 (연구윤리 준수)
1. 표절 금지
2. 데이터 조작 금지
3. 중복게재 금지
4. 저자 자격 준수

### 제17조 (윤리위반 처리)
1. 경고: 1회 위반
2. 연구 무효: 중대 위반
3. 징계 회부: 고의적 위반

### 제18조 (실험안전)
1. 안전교육 이수 필수
2. 위험물질 취급 시 교사 감독
3. 안전사고 즉시 보고

## 제7장 예산 및 지원

### 제19조 (연구비 지원)
| 연구 유형 | 지원 한도 |
|----------|----------|
| 기초연구 | 50만원/팀 |
| 심화연구 | 100만원/팀 |
| 졸업논문 | 150만원/인 |

### 제20조 (연구비 사용)
1. 재료비, 시약비
2. 도서구입비
3. 외부기관 분석비
4. 학술대회 참가비

### 제21조 (정산)
1. 증빙서류 제출 필수
2. 연구 종료 후 1개월 내 정산
3. 미정산 시 다음 연구 지원 제한

## 부칙

이 지침은 2024년 3월 1일부터 시행한다.
