# NEIS 자율연구 기재 가이드

## 기재 영역
**창의적 체험활동상황 > 자율활동 또는 진로활동**

---

## 기재 원칙

### 1. 기본 원칙
- 객관적 사실에 기반하여 기재
- 학생 개인의 역할과 성장을 구체적으로 서술
- 정량적 결과보다 연구 과정과 역량 발달 중심
- 과장이나 미화 없이 실제 활동 기술

### 2. 기재 시 유의사항
- 팀 연구의 경우 개인별 역할 명시 필수
- 연구 주제는 정확하게 기재
- 외부 기관 연계 시 기관명 포함 가능
- 수상 실적은 별도 영역에 기재 (자율연구 기재란 X)

---

## 기재 예시

### 기초 자율연구 (1학년)

**예시 1: 실험 중심 연구**
```
'(연구주제)' 연구에서 (역할: 팀장/실험 담당/데이터 분석 담당 등)을 맡아
(구체적 활동: 실험 설계, 데이터 수집, 결과 분석 등)을 수행함.
(역량 발현: 과학적 탐구 능력, 문제 해결력, 협업 능력 등)을 보이며
(성장 포인트: 연구 방법론 습득, 논리적 사고력 향상 등)함.
```

**예시 2: 이론 중심 연구**
```
'(연구주제)' 연구에 참여하여 (역할)을 담당함.
선행 연구 분석을 통해 (발견/이해한 내용)을 파악하고,
이를 바탕으로 (수행한 연구 활동)을 진행함.
연구 과정에서 (발휘한 역량)을 보임.
```

### 심화 자율연구 (2학년)

**예시 1: 외부 기관 연계**
```
(외부기관명)과 연계한 '(연구주제)' 심화연구에서 (역할)을 수행함.
(연구 방법: 시뮬레이션, 현장 실험 등)을 활용하여 (연구 내용)을 탐구함.
연구 결과 (발견/결론)을 도출하였으며, (학술대회/발표회)에서 연구 성과를 공유함.
이 과정에서 (심화된 역량: 전문 연구 수행 능력, 학문적 의사소통 능력 등)을 함양함.
```

**예시 2: 창의재단 R&E**
```
한국과학창의재단 R&E 사업 '(연구주제)' 연구팀의 (역할)으로 참여함.
(대학/연구기관)의 (연구책임자) 교수 지도 하에 (연구 활동 내용)을 수행함.
(연구 과정에서의 성장 및 역량)을 보임.
```

### 졸업논문연구 (3학년)

**예시**
```
졸업논문 '(논문 제목)' 연구를 수행함.
(연구 주제 선정 배경)에서 출발하여 (연구 방법론)을 적용해 (연구 내용)을 탐구함.
연구 결과 (주요 발견/결론)을 도출하였으며,
구술심사에서 (심사 과정에서 보인 역량)을 발휘함.
3년간의 자율연구 경험을 종합하여 (학문적 성장)을 이룸.
```

## 기재 체크리스트

- [ ] 연구 주제가 정확하게 기재되었는가?
- [ ] 학생 개인의 역할이 명시되었는가?
- [ ] 구체적인 연구 활동이 서술되었는가?
- [ ] 역량 및 성장 포인트가 포함되었는가?
- [ ] 객관적 사실에 기반하여 작성되었는가?
- [ ] 글자 수 제한을 준수하였는가? (영역별 상이)

---

## 주의사항

### 기재 불가 항목
- 교외 수상 실적
- 논문 게재 여부 (학술지명 등)
- 구체적인 순위나 등수
- 타 학생과의 비교 표현

### 권장하지 않는 표현
- "우수한", "뛰어난" 등 평가적 표현
- "최초로", "유일하게" 등 과장 표현
- 연구 결과의 정량적 수치 나열
